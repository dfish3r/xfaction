xfaction - Private Room Help

Xfaction allows any user to create a private chat room.
What features are supported in private chat rooms?

  • Persistent Rooms: you can control whether your room will continue to exist even if everyone leaves
  • Hidden Rooms: you can control whether your room will display in the list of available rooms
  • Members Only Rooms: you can control who has access to your room
  • Moderated Rooms: you can control who moderates, who can speak, and who can only listen
See the instructions below for how to create and configure a private room.

Pidgin

Launch the Pidgin application and login
Select Buddies » Join a Chat, type the name of the room you wish to create, and then click 'Join'
Select 'Configure Room' to set your room's configuration
Configure your room anyway you want
To enable public logging: Select Options » 'Enable Logging'
If you ever need to change your room configuration, type '/config' in the message box
Invite uesrs to your room: Select Conversation » Add
To moderate your room you use the '/role' command in the message box
Description Command
Make someone else a moderator: 
/role character-realm moderator
Allow a user to talk: 
/role character-realm participant
Allow a user to listen: 
/role character-realm vistor
Kick a user: 
/role character-realm none
or 
/kick character-realm
To add additional admins you use the '/affiliate' command in the message box
Description Command
Make someone else an admin: 
/affiliate character-realm admin
Make someone else an owner: 
/affiliate character-realm owner
For help with any command, type '/help' in the message box
