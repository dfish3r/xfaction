xfaction - Help

Xfaction can be accessed from either a web browser or an instant messaging client. The help below covers both approaches but if you are using Xfaction for the first time and have not yet created an account you should log in and register your account with a web browser before using an instant messenger client. This will allow you to review our acceptable use policy.

Web Browser

Go to http://www.xfaction.net/
Enter your information, if you've never been here check "Register New Account"
A buddy list should pop up
Join a room by clicking on the drop down menu "Click to Start..."
Select a room to join
You're in the room now!
Say something!
Set your status as you move from Battleground to Battleground


Pidgin

Register an account using the web interface, see above for details.
Launch the Pidgin application
Add your Xfaction account to Pidgin: Select Accounts » Manage Accounts » Add
Accept the xfaction certificate
Enter your password
Select Buddies » Join a Chat, then click the 'Room List' button
Choose a room and click 'Join'
Say something
Click the 'Available' drop down and select 'New status', add new status settings for the battlegrounds you participate in. Select the appropriate status when you enter a battleground.


Adium

Register an account using the web interface, see above for details.
Launch the Adium application and add your Xfaction account to Adium: Select File » Add Account » Jabber
Accept the xfaction certificate
Enter your password
Select File » xfaction account » Discovery Browser, then expand the Chatrooms listing
Choose a room and click 'Join'
Say something
Click the 'Status' drop down and select 'Edit Status Menu...', add new status settings for the battlegrounds you participate in. Select the appropriate status when you enter a battleground.


Gajim

Register an account using the web interface, see above for details.
Launch the Gajim application
Add your Xfaction account to Gajim: Select Edit » Accounts » Add
Enter your password
Accept the xfaction certificate
Select Actions » Discover Services, then click the 'Chatrooms' and 'Browse'
Choose a room and click 'Join'
Say something
Click the 'Available' drop down and select 'New status', add new status settings for the battlegrounds you participate in. Select the appropriate status when you enter a battleground.


iChat

iChat offers minimal jabber compatibility and does not support the same level of functionality that other jabber clients have.

Register an account using the web interface, see above for details.
Launch the iChat application and add your Xfaction account to iChat: Select iChat » Preferences » (+) symbol in the bottom left
Accept the xfaction certificate
Select File » 'Go to Chat Room...', then enter the name of the room you would like to join. (iChat does not support room discovery, you must know the name of the room.)
Say something
Click the 'Available' drop down and select 'Edit Status Menu...', add new status settings for the battlegrounds you participate in. Select the appropriate status when you enter a battleground.
